## What is OOP (Object-Oriented Programming)?
Object-Oriented Programming (also known and abbreviated as OOP) are a computer programming paradigm based on the concept of object. An object is a "variable" that is made from a class (also known as instantiaion) that has an attribute and methods.

According to [Wikipedia](https://en.wikipedia.org/wiki/Object-oriented_programming) an object oriented programming is a computer programming paradigm based on the concept of objects, which can contain data and code: data in the form of field (often known as attributes or properties), and code in the form of procedure (often known as methods). In OOP, computer programms are designed by making them out objects that interact with one another.
