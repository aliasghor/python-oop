<h2>What is OOP (Object-Oriented Programming)?</h2>
<p>According to <a href="https://en.wikipedia.org/wiki/Object-oriented_programming">wikikepdia</a>, OOP (also known and abbreviated as OOP) is a programming paradigm based on the concept of objects, which can contain data and code: data in the form of fields (known as attribute or property) and code in the form of procedures (often known as methods). In OOP, computer programs are designed by making them out of objects that interact wuth one another.</p>
